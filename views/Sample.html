Conference Planning - Angular Component

Here, a sample of the conference-planning angular component.

Rendering :

{{ event.title }}
{{ event.speaker }} - {{ event.type }} - {{ placeMapping[event.place] }}

JS

Concerning event definition, no configuration is required, different dates or places are computed from provided data :

HTML :

Define an event HTML template as angular transclusion and access the current event with the scope attribute named "event" : 

      <conference-planning events="eventsList" options="options">
        <event>
            <span class="title">{ event.title }</span>
            <span class="speaker">{ event.speaker }</span>
        </event>
      </conference-planning>

CSS :

From the outside of the directive, you can customize the whole event element by overloading the css class 'event' : 
      .hands-on {
        background-color: cornflowerblue;
      }

      .university {
        background-color: #eb9316;
      }

      .quickie {
        background-color: forestgreen;
      }

      .event .title {
        padding-left: 15px;
        font-weight: bold;
      }

      .event .speaker {
        padding-left: 15px;
      }
